# සෙන (Sena) Project Starter

## Getting Started
1. Complete setup:
```bash
chmod +x setup-project.sh
./setup-project.sh
```

2. Create GitHub repository:
```bash
./utils/github-setup.sh
```

3. Start development server:
```bash
npm run dev
```
